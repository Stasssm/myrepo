Подтягивания широким хватом за голову — техника выполнения упражнения:

  1. Схватитесь за перекладину турника широким хватом. Ноги согнуты в коленях и скрещены . Вы можете использовать перчатки для снятия нагрузки с кистей.
  2. Без резких движений подтяните тело вверх. Крайнее положение — шея касается перекладины.
  3. Медленно опуститесь в исходное положение.
  4. Вы можете использовать отягощение. Для этого необходимо использовать пояс с отягощением. Нельзя закреплять отягощение на ногах!
