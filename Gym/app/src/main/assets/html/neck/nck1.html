  • Тип упражнения: Изолирующее
  • Вид упражнения: Силовое
  • Тип оборудования: Другое
  • Тип усилия: Другое
  • Уровень сложности: Средний

Поднятие диска, лежа головой вверх на скамье — техника выполнения упражнения:

  1. Лягте головой вверх на скамью. Край скамьи должен проходить по линии лопаток – это необходимо для достижения наибольшей эффективности упражнения.
  2. Диск должен находиться на лбу, придерживайте его руками. Рекомендуем начать выполнение упражнения с диском весом 2,5 кг и увеличивать вес по мере укрепления мышц шеи.
  3. На вдохе опустите голову вниз.
  4. На выдохе поднимите голову вверх, чуть выше среднего положения.
  5. Выполняйте данное упражнение медленно, без резких движений.
