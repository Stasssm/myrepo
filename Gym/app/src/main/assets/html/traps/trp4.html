• Целевые мышцы: Трапеции
  • Тип упражнения: Изолирующее
  • Дополнительные мышцы: Плечи, Средняя часть спины
  • Вид упражнения: Силовое
  • Тип оборудования: Машина Смита
  • Тип усилия: Тяга
  • Уровень сложности: Начинающий

Шраги в машине Смита — техника выполнения упражнения:

  1. Для начала установите гриф в тренажере Смита в удобное для вас положение. Обхватите гриф по ширине плеч, ладони направлены вниз.
  2. Снимите гриф с тренажера. Встаньте прямо, расслабьте плечи и руки. Спина прямая. Это будет вашим исходным положением.
  3. На вдохе поднимите гриф как можно выше.  Удерживайте плечи в крайнем верхнем положении в течение 1-2 секунд. Подсказка: старайтесь не задействовать бицепсы при выполнении упражнения. Работать должны только плечи.
  4. Опустите гриф в исходное положение.
